# Keylord

Password manager built in Express

```
cd Keylord
npm -install
```

You need to set up your database using the **keylord_structure.sql** file and creating a **.env** file in the root of the application with the following format
```
DB_USER="<username>"
DB_PWD="<password>"
DB_NAME="<db_name>"
```

Use the following command to start the app
```
nmp start
```

You will find the app running at **localhost:3000**
